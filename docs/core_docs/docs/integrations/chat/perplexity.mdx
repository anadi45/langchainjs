# Perplexity

## Setup

1. Create a Perplexity account and get your API key [here](https://www.perplexity.ai/settings/api).
2. Export or set your API key inline. The ChatPerplexity class defaults to process.env.PERPLEXITY_API_KEY.

```bash
export PERPLEXITY_API_KEY=your-api-key
```
